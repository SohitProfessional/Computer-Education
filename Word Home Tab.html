Clipboard Group (क्लिपबोर्ड ग्रुप):

  • Cut (कट)
  • Copy (कॉपी)
  • Paste (पेस्ट)
  • Format Painter (फ़ॉर्मेट पेंटर)

1. Cut (कट) [ Ctrl + X ]:

जब किसी शब्द (Text), वाक्य (Sentence) या चित्र (Image) को एक स्थान से हटाकर दूसरे स्थान पर ले जाना हो, तो कट (Cut) विकल्प (Option) का उपयोग किया जाता है। Cut करने पर चयनित (Selected) शब्द (Text), वाक्य (Sentence) या चित्र (Image) अपने मूल स्थान (Original Location) से हट जाता है और अस्थायी (Temporary) रूप से क्लिपबोर्ड (Clipboard) में सुरक्षित हो जाता है। इसके बाद आवश्यकता अनुसार उसे किसी भी स्थान पर पेस्ट (Paste) किया जा सकता है।

Shortcut Key: Ctrl + X

यह शॉर्टकट कुंजी कीबोर्ड के माध्यम से कट (Cut) विकल्प को शीघ्रता (Quickly) से लागू करने के लिए उपयोग की जाती है।

👉 उदाहरण (Example): यदि आपने एक पैराग्राफ गलत स्थान पर लिख दिया है, तो उसे कट (Cut) करके सही स्थान पर पेस्ट (Paste) कर सकते हैं।

2. Copy (कॉपी) [ Ctrl + C ]:

जब किसी शब्द (Text), वाक्य (Sentence) या चित्र (Image) को उसकी मूल जगह पर रखते हुए उसकी एक प्रति (Duplicate) बनानी हो, तो कॉपी (Copy) विकल्प (Option) का उपयोग किया जाता है। Copy करने पर चयनित (Selected) शब्द (Text), वाक्य (Sentence) या चित्र (Image) अपने मूल स्थान (Original Location) पर ही रहता है और उसकी एक प्रति अस्थायी (Temporary) रूप से क्लिपबोर्ड (Clipboard) में सुरक्षित हो जाती है। इसके बाद आवश्यकता अनुसार उसे किसी भी स्थान पर पेस्ट (Paste) किया जा सकता है।

Shortcut Key: Ctrl + C

यह शॉर्टकट कुंजी कीबोर्ड के माध्यम से कॉपी (Copy) विकल्प को शीघ्रता (Quickly) से लागू करने के लिए उपयोग की जाती है।

👉 उदाहरण (Example): यदि आप किसी पैराग्राफ की एक प्रति बनाकर उसे दस्तावेज़ (Document) के दूसरे भाग में रखना चाहते हैं, तो उसे कॉपी (Copy) करके इच्छित स्थान पर पेस्ट (Paste) कर सकते हैं।

3. Paste (पेस्ट) [ Ctrl + V ]:

जब कट (Cut) या कॉपी (Copy) किए गए शब्द (Text), वाक्य (Sentence) या चित्र (Image) को किसी स्थान पर जोड़ना (Insert) हो, तो पेस्ट (Paste) विकल्प का उपयोग किया जाता है। Paste करने पर क्लिपबोर्ड (Clipboard) में सुरक्षित सामग्री चयनित स्थान पर प्रदर्शित (Display) हो जाती है।

Shortcut Key: Ctrl + V

यह शॉर्टकट कुंजी कीबोर्ड के माध्यम से पेस्ट (Paste) विकल्प को शीघ्रता (Quickly) से लागू करने के लिए उपयोग की जाती है।

👉 उदाहरण (Example): यदि आपने किसी पैराग्राफ को कॉपी (Copy) या कट (Cut) किया है, तो उसे दस्तावेज़ (Document) के इच्छित स्थान पर पेस्ट (Paste) कर सकते हैं।

4. Format Painter (फॉर्मेट पेंटर)

जब किसी शब्द (Text), वाक्य (Sentence) या पैराग्राफ (Paragraph) की स्वरूपण शैली (Formatting Style) को उसी प्रकार किसी अन्य टेक्स्ट पर लागू करना हो, तो फॉर्मेट पेंटर (Format Painter) विकल्प का उपयोग किया जाता है।

Format Painter का उपयोग करने पर चयनित टेक्स्ट की फ़ॉन्ट शैली (Font Style), आकार (Size), रंग (Color), बोल्ड (Bold), इटैलिक (Italic), अंडरलाइन (Underline) आदि सभी स्वरूपण गुण कॉपी हो जाते हैं।

Shortcut Key:
Copy Format: Ctrl + Alt + C
Paste Format: Ctrl + Alt + V

उपयोग विधि:
1. टेक्स्ट चयनित करें।
2. Format Painter पर क्लिक करें।
3. दूसरे टेक्स्ट पर लागू करें।

👉 उदाहरण: किसी Heading का फॉर्मेट कई जगह लागू करने के लिए Format Painter का उपयोग किया जा सकता है।

2. Font Group (फॉन्ट ग्रुप):

Microsoft Word में Font Group बहुत महत्वपूर्ण भाग है। यह Home Tab के अंदर होता है और इसका मुख्य काम टेक्स्ट (Text) की डिज़ाइन, स्टाइल और फॉर्मेटिंग को बदलना है।
जब हम किसी भी डॉक्यूमेंट में कुछ लिखते हैं, तो वह साधारण (Normal) दिखाई देता है। लेकिन अगर हमें उसे आकर्षक (Attractive), स्पष्ट (Clear) और प्रोफेशनल (Professional) बनाना है, तो हम Font Group का उपयोग करते हैं।

(i). Font (फॉन्ट):

जब किसी दस्तावेज़ (Document) में लिखे गए शब्द (Text) की लिखावट (Style) बदलनी हो, तो फॉन्ट (Font) विकल्प (Option) का उपयोग किया जाता है। फॉन्ट बदलने से अक्षरों की डिज़ाइन (Design) और स्वरूप (Appearance) बदल जाता है, जिससे दस्तावेज़ अधिक आकर्षक (Attractive) और पढ़ने में आसान (Readable) बनता है। प्रत्येक फॉन्ट की अपनी अलग शैली होती है, जैसे – Arial, Mangal, Calibri, Times New Roman आदि।

फॉन्ट बदलने पर चयनित (Selected) टेक्स्ट (Text) की लिखावट बदल जाती है, लेकिन उसका मूल अर्थ (Meaning) नहीं बदलता। यह केवल टेक्स्ट की प्रस्तुति (Presentation) को बेहतर बनाता है।

स्थान (Location): Home Tab → Font Group → Font

👉 उदाहरण (Example): यदि आप किसी शीर्षक (Heading) को साधारण लिखावट की जगह अधिक प्रभावशाली बनाना चाहते हैं, तो उसे Select करके उसका फॉन्ट (Font) बदल सकते हैं, जैसे Calibri से Times New Roman।

(ii).Bold (बोल्ड) [Ctrl + B]:

जब किसी शब्द (Text), वाक्य (Sentence) या पैराग्राफ (Paragraph) को गहरा (Dark) और मोटा (Thick) दिखाना हो, तो Bold विकल्प (Option) का उपयोग किया जाता है। Bold करने से चयनित (Selected) टेक्स्ट (Text) अधिक स्पष्ट (Clear) और प्रभावशाली (Impressive) दिखाई देता है। इसका उपयोग महत्वपूर्ण (Important) शब्दों, शीर्षक (Heading) या विशेष जानकारी को उभारने (Highlight) के लिए किया जाता है।

स्थान (Location): Home Tab → Font Group → Bold (B)
Shortcut Key: Ctrl + B

यह शॉर्टकट कुंजी कीबोर्ड के माध्यम से Bold विकल्प को शीघ्रता (Quickly) से लागू करने के लिए उपयोग की जाती है।

👉 उदाहरण (Example): यदि आप किसी आवेदन पत्र (Application) में अपना नाम या महत्वपूर्ण तिथि (Important Date) को अलग से दिखाना चाहते हैं, तो उसे Select करके Bold कर सकते हैं।
